## 🛠️ How to Use

### For Visitors:
1. Visit [researchhub5.github.io](https://researchhub5.github.io)
2. Search for resources using the search bar
3. Click category filters to narrow down results
4. Click **Download** to get any resource

### For Contributors (Adding Files):
1. Go to the repository
2. Click **"Add file"** → **"Upload files"**
3. Upload your research file
4. Update the `files` array in `index.html`
5. Commit changes
